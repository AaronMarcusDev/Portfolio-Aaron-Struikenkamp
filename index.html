3D Flocking Bird Game

What: An interactive 3D ecosystem where birds flock together using Craig Reynolds' flocking rules, fully implemented in three dimensions. The player can throw rocks at the birds, spawn new birds, and regenerate procedurally generated Perlin noise terrain.

My role: Full implementation — designed and programmed all classes including Bird, Rock, Terrain, ParticleSystem, and Menu.

Key features:

  • 3D flocking with separation, alignment, and cohesion rules
  • Gaussian distribution for natural bird spawning
  • Perlin noise terrain generation with adjustable height
  • Physics-based rock throwing with gravity
  • Particle explosion system on collision
  • Interactive menu for terrain height control

Technologies used: Processing, PVector, Perlin noise, 3D transformations, Euler integration

What I learned: Advanced 3D programming, flocking algorithms, particle systems, procedural terrain generation, and structuring complex programs with multiple interacting classes.
