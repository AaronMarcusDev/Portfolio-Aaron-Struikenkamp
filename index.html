Ultrasonic Flappy Bird

Module: Module 2 — Programming & Physical Computing

Made with: Hugo Otten

What: A hands-free reimagining of the classic Flappy Bird game. An HC-SR04 ultrasonic sensor connected to an Arduino Uno detects hand gestures — wave your hand close to make the bird jump. The controller wirelessly communicates with the Processing game client over the internet. A piezoelectric buzzer provides audio feedback for jumps and game events. The controller housing is made from recycled tea packaging!

Tech stack: Arduino Uno, HC-SR04 ultrasonic sensor, piezoelectric buzzer, Processing (game server + client), custom serial/network communication protocol.

My role: Programmer & Hardware

What I learned: Physical computing, serial communication, client-server architecture, separating concerns between hardware sensing, communication middleware, and game logic.
